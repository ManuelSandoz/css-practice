Disease Spread Tracking Aplication (Distra)

Distra is a mobile app built using React Native with the goal of tracking the spread of the covid_19 pandemic. The application gave users up to date inforamtion on the state of the pandemic. For example number of cases, number of deaths and information on stepts to take to prevent contagion of the disease. The aplication also gathered voluntary demographic information from the users which allowed the development team to create a data set. This data set could be used to study the behaviour of this disease and any possible patterns of spreading.

As a developer for this project I worked on the design of the UI of the project as well as the creation of the project. Specifically I built the navigation components for the aplication as well as created the map view which gave the user a visual representation of the state of the pandemic. I was also in charge of integrating a data base for this project in order to give the application data permanence. For this I integrated firebase into the project which gave the application a quick and secure data base while also providing extra features like account creationd using google accounts.
